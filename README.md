# LVK Cumulative Detection Plot with Lensing Forecast, O1 – O4c

## Introduction
This repository provides Python scripts for generating cumulative detection plots for gravitational-wave events identified by the LIGO–Virgo–KAGRA (LVK) collaborations across observing runs O1 through O4c, extended to include a gravitational lensing event forecast.

The base plotting framework is adapted from the official LVK cumulative-event plotting scripts developed and maintained at
https://git.ligo.org/operations/cumulative-event-plot.
